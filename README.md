# ALICE

Astral Local Intelligence and Content Engine.

This is a standalone Next.js project scaffold for the ALICE concept: brochure ingestion, local SEO content generation, keyword intelligence, GBP workflows, and reputation operations.

## Run locally

1. `npm install`
2. `npm run dev`

## Key files

- `app/page.tsx`: ALICE landing and project framing page
- `app/globals.css`: shared styling for the project surface
- `docs/alice-project-brief.md`: concise project brief and V1 scope
